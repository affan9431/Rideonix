# 🚖 Rideonix - Full Stack Ride Booking App

RideOnEx is a ride-booking web application that connects riders with nearby drivers. It features real-time location updates, OTP-based login, secure ride handling, and a built-in review system.

---

## 🚀 Tech Stack

### 🔧 Frontend
- React.js
- Axios
- Tailwind CSS
- Socket.io-client
- React Router DOM
- Toastify

### 🛠️ Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- Socket.io
- JWT for auth
- Fast2SMS (for OTP)
- Bcrypt (for hashing)
- CORS

---

## 🔐 Security Implementations

- Rate Limiter
- XSS
- Helmet
- MongoSanitize

---

## 📦 Getting Started

### 🔧 Backend Setup (Flask)
```bash
cd back-end
npm install
npm start
```

### 🔧 Frontend Setup (React + Vite)
```bash
cd front-end
npm install
npm run dev
```
